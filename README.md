## Getting Started

1. Clone the repo

   ```sh
   git clone https://github.com/omardoescode/minesweeper
   ```

2. Install prerequisites

- pip
  ```sh
  pip install -r requirements.txt
  ```

## Usage

- To start the GUI version

  ```sh
  python main_gui.py
  ```

- To start the CLI version
  ```sh
  python main_cli.py
  ```
